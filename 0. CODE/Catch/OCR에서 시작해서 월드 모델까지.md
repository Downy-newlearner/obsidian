손글씨를 포함한 현실의 아날로그 텍스트를 디지털 텍스트로 변환하는 OCR기술은 현재 '자동 채점 프로그램'의 구현 과정에서 많이 발전할 예정이다.
- 이는 월드 모델 제작에 일조해서 다음과 같은 고민으로 이어질 수 있다.
	- 시각 정보에서 어떤 정보를 가져와야하는지 정의
		- 사용자가 해당 공간에서 어떤 행위를 하려고 하는지에 대한 '의도'가 파악되어야 '어떤 정보'를 제공할지 결정할 수 있다.
		- 이 때 '의도'는 사용자의 =='행동'의 맥락==을 통해 파악할 수 있다.
			- 예를 들어 사용자가 집 -> 도서관 -> 카페를 갔다면 카페에서는 도서관에서 공부에 집중하기 위한 카페인 음료를 사러 갔다는 사용자의 의도를 추측할 수 있다.(행동의 맥락 -> 의도 추측)
			- 그러므로 해당 카페에서 행사중인 카페인 음료에 대한 정보 또는 카페인 음료들에 카페인 함량에 대한 정보를 제공할 수 있다.(의도 -> 정보 제공)
			- 또는 사용자의 카페인 민감도에 대한 정보가 존재한다면 사용자에게 음료 추천을 할 수도 있다.(개인화)
