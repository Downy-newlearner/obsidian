**React Router**

> “React 앱 안에서 여러 페이지처럼 보이게 해주는 도구”예요.

### **예시:**

- `/` → 홈 페이지
- `/about` → 소개 페이지
- `/profile` → 내 프로필 페이지

React는 원래 한 페이지로 구성된 앱([[SPA]])입니다. **“주소(URL)에 따라 화면을 바꾸는 기능”**을 React Router가 제공합니다.

## React Router는 설치해야한다.
```
cd quote-generator && npm install react-router-dom @types/react-router-dom
```
