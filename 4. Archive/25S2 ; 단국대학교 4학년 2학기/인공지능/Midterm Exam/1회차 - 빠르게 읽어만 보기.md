#### 8. lec03-game-tree
- Min-Max 알고리즘
	```python
	
	function minimax(node, depth, isMaxPlayer):
		if depth == 0 or (node가 단말노드):
			return node's heuristic value
			
		if ismaxPlayer:
			value = -∞
			for each child of node:
				value = max(value, minimax(child, depth-1, False))
			return value
			
		else
			value = +∞
			for each child of node:
				value = min(value, minimax(child, depth-1, True))
			return value
	
	```
