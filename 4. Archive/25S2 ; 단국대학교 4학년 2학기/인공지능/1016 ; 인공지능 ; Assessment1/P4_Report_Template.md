## 1. 문제 개요

- 가중치 그래프 G50w.csv(50×50) 기반의 경로 탐색
- Dijkstra와 A\* 구현 및 최단/최장 경로 결과 비교

## 2. 알고리즘 요약

- Dijkstra: 비음수 가중치 그래프에서 최단 경로를 보장
- A\*: 휴리스틱을 사용한 탐색 효율 개선, 본 과제의 h는 구간별 난수 기반

## 3. 휴리스틱 정의 및 코드

```python
import numpy as np
np.random.seed(42)  # 재현성
h = np.zeros(50)
h[0:20] = 40 - 40 * np.random.rand(20)
h[20:40] = 30 - 30 * np.random.rand(20)
h[40:50] = 20 - 10 * np.random.rand(10)
print(h[:10])
```

## 4. 주요 코드 설명

- 그래프 로드: CSV → 인접 리스트 변환(0은 간선 없음)
- Dijkstra: 우선순위 큐를 이용한 최단 거리 갱신
- A\*: f=g+h 기반의 탐색, h는 과제 명세대로 생성
- 최장 경로: 전체 간선 가중치 음수화 후 동일 알고리즘 재사용
