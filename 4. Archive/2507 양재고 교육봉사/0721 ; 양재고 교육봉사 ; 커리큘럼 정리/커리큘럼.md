### **[총 주제]**

  

**“GPT와 함께하는 인공지능 체험 수업 – 프롬프트 하나로 세상을 바꿔볼까?”**

---

## **1교시: LLM이 뭐고, 왜 다들 난리야? (이론+체험, 50분)**

  

### **● 목표: LLM 개념 이해 및 프롬프트에 따른 결과 차이 체험**

| **시간** | **내용**                                                                               |
| ------ | ------------------------------------------------------------------------------------ |
| 0~10분  | Ice-breaking: “이 문장 누가 썼을까?” (GPT가 만든 문장 vs 사람) 퀴즈 - 자소서 예시                          |
| 10~20분 | **LLM이란?** 개념 설명• 챗GPT는 어떻게 만들어졌을까?• 언어모델 vs 챗봇 차이• GPT, 제미나이 등 모델 간 차이              |
| 20~30분 | **프롬프트 실험** 간단한 질문을 여러 방식으로 바꿔서 GPT에 입력해보기예: “날씨 어때?” → “서울 비와?” → “오늘 서울에 비 올 확률은?” |
| 30~45분 | **결과 비교 실습**• 학생들이 조별로 질문 만들고, 다양한 톤/방식으로 GPT에게 물어본 뒤 응답 비교• 포스트잇에 가장 창의적인 결과 공유     |
| 45~50분 | 정리 및 질의응답“LLM은 프롬프트에 민감한 똑똑한 앵무새다!” 요약                                               |

---

## **2교시: 너도 할 수 있어! 프롬프트 엔지니어링 체험 (실습 중심, 50분)**

  

### **● 목표: zero-shot / few-shot 기법을 실습하고, ‘잘 쓰는 법’을 익힘**

| **시간** | **내용**                                                                                                    |
| ------ | --------------------------------------------------------------------------------------------------------- |
| 0~10분  | **프롬프트 엔지니어링 소개**• zero-shot, few-shot 개념• 실생활 예시: “이 문장 맞춤법 고쳐줘”, “이 글 요약해줘”                             |
| 10~30분 | **조별 실습 ①: GPT로 미션 수행**• 미션 카드 배부 (예: “수학문제 풀이하는 GPT 만들기”, “글 요약 봇 만들기”)• zero-shot → few-shot 방식으로 바꿔 실험 |
| 30~40분 | **조별 발표**: 어떤 프롬프트가 가장 효과적이었는지 발표                                                                         |
| 40~48분 | **solar-prompt-cookbook 예시 공유**• 실제 데이터톤에서 쓰인 고급 프롬프트 예시 소개                                               |
| 48~50분 | “좋은 질문이 좋은 답을 만든다” 정리 & 피드백                                                                               |

---

## **3교시: 고등학생도 할 수 있다! GPT로 할 수 있는 일들 (활용 체험 중심, 50분)**

  

### **● 목표: 실제 학교생활에서 쓸 수 있는 GPT 활용법 체험**

| **시간** | **내용**                                                                    |
| ------ | ------------------------------------------------------------------------- |
| 0~10분  | **활용 사례 소개**• 작문 도와주는 GPT• 영어 면접 준비• 과제 요약, 시간표 짜기                        |
| 10~35분 | **미니 프로젝트 실습** (조별 선택)① 자기소개서 작성② 영어 스피킹 연습③ 나만의 공부 도우미 만들기④ 학교 생활 Q&A 챗봇 |
| 35~45분 | **GPT와 협업한 결과물 발표**                                                       |
| 45~50분 | 마무리 퀴즈 (수업 전체 복습) & 만족도 조사                                                |
