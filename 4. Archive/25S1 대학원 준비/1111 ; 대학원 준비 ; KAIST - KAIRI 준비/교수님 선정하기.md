## KAIST 김재철AI대학원 교수진 연구분야 정리

### 김기응 교수 (Kee-Eung Kim)
- 오프라인 강화학습 (Offline reinforcement learning)[3]
- 대규모 언어 모델의 새로운 응용 (Large-scale language models for novel applications)[3]
- 순차적 의사결정 문제 (마르코프 의사결정 과정, POMDP)[1]


### 김승룡 교수 (Seungryong Kim)
- 제어 가능한 다시점 비디오 생성 모델
- 텍스트-이미지/비디오 개인화
- 확산 샘플링 가이던스
- 3D/4D 파운데이션 모델
- 비디오 포인트 트래킹
- Neural Radiance Fields / Gaussian Splatting


### 신진우 교수 (Jinwoo Shin)
- 비전-언어-행동 모델 (Vision-language-action models)
- LLM 에이전트
- 비디오 확산 및 월드 모델

### 심현정 교수 (Hyunjung Shim)
- 다중 모달 대규모 언어 모델의 편향 완화
- 자율주행 (경로 계획, VQA 등)
- 3D 생성

### 이기민 교수 (Kimin Lee)
- 인간 피드백으로부터의 강화학습 (Reinforcement learning from human feedback)
- 의사결정 에이전트
- AI 정렬 / 안전성 (AI Alignment / Safety)
- 월드 모델

### 이주호 교수 (Juho Lee)
- 통계적 추론을 위한 파운데이션 모델
- 인식적 불확실성의 예측 모델링
- 딥러닝 기반 날씨 예측
- 대규모 언어 모델을 위한 불확실성 정량화
- 불확실성 보정 / 강건한 의사결정 / 베이지안 딥러닝 관련 주제


### 주재걸 교수 (Jaegul Choo)
- 확산 모델 기반 이미지 생성 기술 및 응용 (가상 피팅, 헤어스타일 변환, 얼굴 교체, 머리 교체, 자동 색상화)
- 3D 생성 모델 (예: 인간 머리 아바타)
- 음성 신호 품질 향상 및 노이즈 제거
- LLM 모델의 검색 증강 생성(RAG), 프롬프트 엔지니어링 및 파인튜닝
- 의료 영상 인식 및 합성
- 시계열 예측

### 최재식 교수 (Jaesik Choi)
- 설명 가능한 인공지능 (XAI)
- 시계열 데이터 분석/예측
- 산업 인공지능
- 헬스케어 인공지능

### 황성주 교수 (Sungju Hwang)
- StableDiffusion을 활용한 제어 가능한 비디오 생성
- 디바이스 인식 AutoML (신경망 구조 탐색, 하이퍼파라미터 최적화)
- 대규모 언어 모델의 효율적인 학습 및 추론 (압축, 분산 학습, 추론과 지식의 분리)
- 자기 진화 신경망

**총평**: 김재철AI대학원의 교수진은 강화학습, 대규모 언어 모델(LLM), 컴퓨터 비전(확산 모델, 3D/4D 생성), 로봇공학, 그래프 머신러닝, 헬스케어 AI, AI 이론 등 AI의 거의 모든 핵심 분야를 망라하고 있습니다. 특히 강화학습 관련 교수가 다수이며, 최신 트렌드인 LLM과 확산 모델 연구가 활발합니다.
