# 🧐 Underfitting이란? [Answer Post]

## 형식적 답

Underfitting은 모델이 훈련 데이터에 충분히 학습되지 않아 데이터의 주요 패턴을 제대로 잡아내지 못하는 상태를 말합니다. 이 경우 모델은 훈련 데이터뿐만 아니라 새로운 데이터에 대해서도 정확한 예측을 하지 못하게 됩니다. Underfitting은 보통 모델이 너무 단순하거나, 학습 시간이 충분하지 않거나, 중요한 특징을 놓쳤을 때 발생합니다.

## AI 답변

Underfitting이란 기계 학습 모델이 훈련 데이터에 과도하게 일반화되어 데이터의 패턴과 추세를 제대로 포착하지 못하는 상황을 말합니다. 이는 모델이 너무 단순하거나 데이터를 충분히 학습하지 못한 경우에 발생합니다. Underfitting된 모델은 훈련 데이터뿐만 아니라 새로운 데이터에 대해서도 낮은 성능을 보입니다. 주요 원인과 해결 방법은 다음과 같습니다.

원인:
1. 모델이 너무 단순함
2. 데이터의 특징이 부족함
3. regularization이 과도함

해결 방법:
1. 모델의 복잡도를 높임 (더 많은 파라미터, 깊은 신경망 등)
2. 특징 엔지니어링을 통해 유의미한 특징 추가
3. regularization 하이퍼파라미터 조정
4. 훈련 데이터 증가
5. 모델 아키텍처 변경

Underfitting은 과적합(Overfitting)에 비해 상대적으로 모델 성능 개선의 여지가 크므로, 모델 복잡도를 적절히 높이고 유의미한 특징을 추가하는 등의 방법으로 해결할 수 있습니다.
