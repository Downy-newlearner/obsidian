### 1. 데이터 설정 파일

데이터 설정 파일은 YOLOv7 모델이 데이터를 어떻게 읽고 사용할지를 정의한 YAML 파일입니다. 일반적으로 프로젝트 디렉토리의 데이터 관련 폴더에 위치시킵니다. 예를 들어, `data.yaml`로 이름을 짓고 `./data` 폴더에 넣을 수 있습니다.

#### 데이터 설정 파일 내용

- `train`: 학습 데이터 경로
- `val`: 검증 데이터 경로
- `nc`: 클래스 수
- `names`: 각 클래스의 이름 리스트

#### 예제

```yaml
train: ../AI/dataset/labels/train
val: ../AI/dataset/labels/val
nc: 1
names: ['item']
```
