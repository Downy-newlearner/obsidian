        ![[Source/Untitled 1 49.png|Untitled 1 49.png]]
        
        - JE
            - Jump if Equal
            - 값이 같으면 점프
        - JNE
            - Jump if Not Equal
            - 값이 다르면 점프
        - JG
            - Jump if Greater
            - 왼쪽 인자의 값이 더 크면 점프
            - Destination - Source < 0 이면 점프 (이하 D - S라고 표기)
            - cmpl $3, %ecx는 5-3 > 0 이므로 점프하지 않게 된다.
        - JL
            - Jump if Less
            - 왼쪽 인자의 값이 더 작으면 점프
            - D - S < 0 이면 점프
        - JNG
            - 왼쪽 값이 더 크지 않으면 점프
            - 즉 왼쪽 값이 더 작거나 같으면 점프
        - JNL
            - 왼쪽 값이 더 작지 않으면 점프
            - 즉, 왼쪽 값이 더 크거나 같으면 점프
        - JGE
            - 왼쪽 값이 더 크거나 같으면 점프
        - JLE
            
            - 왼쪽 값이 더 작거나 같으면 점프
