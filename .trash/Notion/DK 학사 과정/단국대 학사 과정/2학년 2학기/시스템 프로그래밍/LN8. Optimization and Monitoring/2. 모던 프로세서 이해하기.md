## 전반적인 특징
- 두 개의 메인 유닛
    1. 명령어 컨트롤
        - 명령어 인출
        - 명령어 해독(오퍼레이션 발생)
    2. 실행 유닛
        - 발생한 오퍼레이션 실행
  
- 기능적 유닛
    - Integer / Branch
    - General Integer
    - FP Add
    - FP Multiplication/Division
    - Load
    - Store
- 슈퍼스칼라
- 무순서 실행
- 분기 예측
- 추정 실행
- 레지스터 리네이밍
