## 브랜치 생성
```
git branch (브랜치명) main
```
- main 브랜치에서 새로운 브랜치를 생성하는 명령어이다.

브랜치들을 자유롭게 이동하면서 해당 브랜치의 코드를 확인하려면 어떻게?

## 브랜치 삭제
```
git branch -D (브랜치명)
```

## 브랜치명 변경
```
git branch -m (브랜치명) (새로운 브랜치명)
```
