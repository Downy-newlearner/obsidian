**Confluence(컨플루언스)**는 **팀 지식관리 및 협업을 위한 문서 작성 도구**로,

주로 팀 내부의 **위키, 문서 협업, 회의록, 프로젝트 계획** 등을 관리할 수 있게 해주는 서비스입니다.

---

## **✅ Confluence란?**

| **항목** | **설명**                               |
| ------ | ------------------------------------ |
| 개발사    | Atlassian (Jira 개발사)                 |
| 형태     | 웹 기반 협업 문서 플랫폼                       |
| 주요 기능  | 실시간 문서 작성/공유, 협업, 페이지 구조화, 검색, 권한 관리 |
| 통합성    | Jira, Trello 등 Atlassian 도구와 원활하게 연동 |

---

## **✅ 어떤 사람들이 쓰나요?**

|**사용자**|**사용 목적**|
|---|---|
|**기술팀 (개발자/기획자)**|기술 문서, API 명세, 회의록 작성, 아키텍처 정리|
|**디자인/PM 팀**|피드백 공유, 스펙 정리, 제품 로드맵 작성|
|**마케팅/운영 팀**|캠페인 플랜, 팀 위키, 콘텐츠 일정 공유|
|**스타트업/중소팀**|팀 위키로 전사 문서 체계 관리|

---

## **✅ 4명 이하 소규모 팀에도 적합한가?**

  

**매우 적합합니다. 이유는 다음과 같아요:**

| **이유**              | **설명**                               |
| ------------------- | ------------------------------------ |
| **정보 정리 구조화**       | 페이지/서브페이지로 체계적인 지식 관리 가능             |
| **간편한 협업**          | 댓글, 멘션, 버전 관리로 커뮤니케이션 효율 ↑           |
| **무료 요금제 제공**       | **10인 이하까지 무료**로 사용 가능 (기능 제한 거의 없음) |
| **스프레드시트보다 확장성 높음** | Notion보다 더 위키 중심적이고 검색/정렬 강력         |

---

## **✅ Confluence vs 다른 도구 (Notion 등)**

|**항목**|**Confluence**|**Notion**|
|---|---|---|
|팀 협업|강력 (Jira 등과 연동 탁월)|유연함 (개인/소규모 팀 친화적)|
|구조화|위키 기반 계층적 페이지|블록 기반 자유로운 구성|
|확장성|엔터프라이즈에 최적화|스타트업, 개인에 최적화|
|검색/접근 권한|세밀한 설정 가능|간단한 설정 위주|
|가격|10인까지 무료|일부 기능 제한 있음 (무료 요금제에서)|

---

## **✅ 요약**

  

> **Confluence는 4인 이하 소규모 팀도 충분히 사용할 수 있는 협업 문서 도구**입니다.

> 특히 **조직적인 지식 공유**나 **회의록/기획 문서 관리**가 필요할 때 매우 유용하고,

> **Jira와 연동해 이슈-문서 트래킹을 통합**하려는 팀에게 최적입니다.
