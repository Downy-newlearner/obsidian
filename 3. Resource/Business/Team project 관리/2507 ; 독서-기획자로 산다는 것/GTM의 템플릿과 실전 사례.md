## ✅ Go-To-Market 전략 템플릿 (Framework)

아래 표는 GTM 전략을 체계적으로 수립하기 위한 템플릿입니다:

|**항목**|**설명**|**예시**|
|---|---|---|
|**1. 목표 시장 (Target Market)**|누구에게 팔 것인가?|20대 대학생, 스타트업 CTO, 소상공인 등|
|**2. 고객 페르소나 (Persona)**|이상적인 고객의 구체적 특성|“김마케팅(34세, SaaS팀장, 문제: 툴 통합 어려움)”|
|**3. 문제 정의 (Pain Point)**|고객이 겪는 문제는?|생산성 도구가 많아 협업이 분산됨|
|**4. 가치 제안 (Value Proposition)**|우리 제품/서비스가 해결하는 방식|하나의 툴로 커뮤니케이션 + 프로젝트 관리|
|**5. 경쟁 분석 (Competitive Landscape)**|경쟁사 비교|Notion, Slack과 차별화된 통합성|
|**6. 유통 채널 (Channels)**|어떤 경로로 도달할 것인가?|SNS, 웹사이트, 커뮤니티, 제휴 채널 등|
|**7. 마케팅 전략 (Marketing Tactics)**|어떤 방식으로 알릴 것인가?|유튜브 리뷰, 인플루언서 체험단, 콘텐츠 마케팅|
|**8. 가격 전략 (Pricing)**|어떤 요금 구조?|무료 체험 → 월 구독제|
|**9. 세일즈 전략 (Sales Strategy)**|어떻게 팔 것인가?|셀프 온보딩 + 고객 성공팀|
|**10. 성과 지표 (KPIs)**|무엇으로 성공을 측정?|CAC, MRR, 전환율, 리텐션율 등|

→ 이 프레임워크를 표로 정리하면, 하나의 **GTM 전략 문서**가 됩니다.

## **✅ 실전 사례 1:** 

## **Notion의 글로벌 확산 전략**

| **항목** | **내용**                         |
| ------ | ------------------------------ |
| 대상     | 스타트업, 프리랜서, 개발자 중심의 글로벌 개인/팀   |
| 가치 제안  | 올인원 문서+데이터베이스 협업 툴             |
| 채널     | SNS, 유튜브, 프로덕트헌트, 커뮤니티(레딧, 슬랙) |
| 세일즈    | **셀프 온보딩 → 자연스러운 팀 확산**        |
| 가격     | 개인 무료, 팀 유료 (프리미엄 전환 유도)       |
| 성과     | 바이럴로 퍼지고, 리텐션/재방문율로 측정         |

→ 핵심: **프리미엄 모델과 바이럴 확산의 조화**

## **✅ 실전 사례 2:** 

## **Slack의 초기 B2B GTM 전략**

| **항목** | **내용**                      |
| ------ | --------------------------- |
| 대상     | 중소기업 팀 단위 사용자               |
| 가치 제안  | 이메일보다 빠른 업무 커뮤니케이션          |
| 채널     | CTO/PM 대상 컨퍼런스, 테크미디어 홍보    |
| 세일즈    | 제품 중심 성장 (PLG) → 팀 초대 방식 유도 |
| 가격     | 일정 기능까지 무료, 이후 사용자 수 기준 과금  |
| 성과     | 조직 단위로 확산, MAU/DAU 추적       |

→ 핵심: **제품 자체가 확산 수단이 되도록 설계 (PLG, Product-Led Growth)**
