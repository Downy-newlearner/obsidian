## 기본 개념

??는 Null-coalescing 연산자
- 왼쪽 값이 null이 아니면 → 왼쪽 값 반환
- 왼쪽 값이 null이면 → 오른쪽 값 반환

```dart
final userId =
    payloadMap['user_id'] ??      // 1순위: 'user_id' 키 확인
    payloadMap['userId'] ??       // 2순위: 'userId' 키 확인
    payloadMap['sub'] ??          // 3순위: 'sub' 키 확인 (JWT 표준)
    payloadMap['id'];             // 4순위: 'id' 키 확인
```
