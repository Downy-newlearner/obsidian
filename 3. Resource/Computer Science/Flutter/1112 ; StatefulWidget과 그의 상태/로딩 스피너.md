앱이나 웹에서 데이터를 불러오거나 작업이 완료될 때까지 기다리는 동안 사용자에게 “지금 로딩 중이에요”라고 시각적으로 알려주는 회전 애니메이션 표시예요.

Flutter에서는 대표적으로 **CircularProgressIndicator()** 위젯이 로딩 스피너 역할을 한다.

## 로딩스피너의 역할 요약
| **항목**            | **설명**                               |
| ----------------- | ------------------------------------ |
| **목적**            | 사용자에게 “지금 백그라운드 작업 중”임을 알려줌          |
| **형태**            | 보통 원이 회전하거나 점이 깜박이는 애니메이션            |
| **Flutter 기본 위젯** | CircularProgressIndicator()          |
| **디자인 변형**        | LinearProgressIndicator() (막대형) 도 있음 |
| **사용 위치**         | API 호출, 로그인 처리, 이미지 로딩, 파일 업로드 등     |
